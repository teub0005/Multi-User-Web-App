Description of Tools

Github

"GitHub is a web-based hosting service for version control using git. It is mostly used for computer code. It offers all of the distributed version control and source code management functionality of Git as well as adding its own features"(Wikipedia). We used github to host our project. We kept all of our files on there as a way to easily update each other.

Bootstrap

"Bootstrap is a free and open-source front-end library for designing websites and web applications. It contains HTML- and CSS-based design templates for typography, forms, buttons, navigation and other interface components, as well as optional JavaScript extensions. Unlike many web frameworks, it concerns itself with front-end development only"(Wikipedia) The templates that bootstrap provided made it excessively easier to format our pages to look appeasing to the user.

JQuery

"jQuery is a cross-platform JavaScript library designed to simplify the client-side scripting of HTML. It is free, open-source software using the permissive MIT License"(Wikipedia). This provided code templates we could use instead of having to develop ourselves.
